# Financial Transactions Data Warehouse

## О проекте

Разработан ETL-пайплайн для международного финтех-сервиса, выполняющего денежные переводы между различными странами и валютами.

Проект решает задачу консолидации транзакционных данных из различных источников, загрузки их в аналитическое хранилище и формирования ежедневной витрины для анализа финансовых показателей компании.

В результате построена инфраструктура, позволяющая автоматически обновлять данные и рассчитывать ключевые бизнес-метрики.

---

## Бизнес-задача

Компания ведёт операции одновременно в нескольких странах, поэтому данные поступают в разных форматах.

Необходимо было:
- собрать данные из источников;
- привести их к единой структуре;
- загрузить в Data Warehouse;
- ежедневно рассчитывать аналитическую витрину для оценки оборота компании.

## Что реализовано

### Загрузка данных

Реализована автоматическая загрузка:
- транзакций пользователей;
- истории курсов валют.

### ETL

Во время загрузки выполняются:
- извлечение данных из PostgreSQL;
- фильтрация завершённых транзакций;
- пакетная загрузка (batch insert);
- сохранение в слой STAGING.

### Хранилище

Спроектированы таблицы:
- transactions
- currencies

Настроены параметры хранения данных:
- "ORDER BY"
- "SEGMENTED BY HASH(...)"
- "оптимизация хранения для аналитических запросов"

### Аналитическая витрина

Построена витрина **global_metrics**, содержащая:
- ежедневный оборот компании;
- количество транзакций;
- число активных пользователей;
- среднее количество операций на аккаунт;
- оборот по валютам в единой валюте.

### Автоматизация

Создан DAG Airflow, который ежедневно:
1. получает новые данные;
2. обновляет таблицы;
3. пересчитывает витрину инкрементально.

## Описание витрины

| Поле                           | Описание                            |
| ------------------------------ | ----------------------------------- |
| date_update                    | дата расчёта                        |
| currency_from                  | код валюты                          |
| amount_total                   | общий оборот в USD                  |
| cnt_transactions               | число транзакций                    |
| avg_transactions_per_account   | среднее число транзакций на аккаунт |
| cnt_accounts_make_transactions | число активных аккаунтов            |
