# Cloud Technologies: DDS & CDM Microservices

## О проекте

В рамках проекта реализованы второй и третий микросервисы конвейера обработки данных для хранилища данных (DWH), построенного по архитектуре Data Vault.

Разработанные сервисы получают сообщения из Apache Kafka, выполняют необходимые преобразования и загружают данные в слои DDS и CDM хранилища данных PostgreSQL.

Для проекта также подготовлены Docker-образы, конфигурация Docker Compose и Kubernetes-манифесты для развёртывания сервисов.

## Что реализовано

### DDS Service

Второй микросервис выполняет следующие задачи:

- получает сообщения из Kafka;
- преобразует входящие данные;
- загружает данные в слой DDS;
- заполняет сущности Hub, Link и Satellite модели Data Vault;
- обеспечивает идемпотентную обработку сообщений;
- публикует обогащённые сообщения в следующий Kafka-топик.

### CDM Service

Третий микросервис:

- получает сообщения из Kafka после обработки DDS;
- использует данные слоя DDS для формирования витрин;
- обновляет агрегированные показатели в слое CDM;
- наполняет пользовательские витрины данными.
