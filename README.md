# Anastasia Ten

## Data Engineer

Привет! Меня зовут Анастасия.

Я начинающий Data Engineer с инженерным образованием и практическим опытом разработки ETL-процессов, построения хранилищ данных, Data Lake и аналитических витрин.

Более 6 лет работаю в ОАО «РЖД», где занимаюсь анализом данных, подготовкой отчетности и сопровождением корпоративных информационных систем. В процессе работы пришла к пониманию, что мне особенно интересны задачи, связанные с обработкой, структурированием и анализом данных. Переход в Data Engineering — осознанное решение, к которому я готовилась больше года.

В 2026 году окончила программу **«Инженер данных»** в Яндекс Практикуме, где выполнила полный цикл учебных проектов: от работы с SQL и проектирования баз данных до разработки распределённых ETL-пайплайнов, потоковой обработки данных и облачных решений.

Сейчас открыта для предложений на позицию **Junior Data Engineer**.
