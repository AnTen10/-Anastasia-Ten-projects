# SWAPI API Client

## О проекте

Реализован клиент для взаимодействия с открытым REST API Star Wars API (SWAPI). Приложение получает список доступных категорий (персонажи, фильмы, планеты и другие сущности), выполняет HTTP-запросы и сохраняет полученные данные в отдельные файлы.

Основная цель проекта — освоить работу с REST API, обработку HTTP-запросов, организацию кода с использованием классов и покрытие функциональности тестами.

---

## Архитектура проекта

```
SWAPI API
    │
    ▼
APIRequester
    │
    ▼
SWRequester
    │
    ▼
save_sw_data()
    │
    ▼
data/
├── films.txt
├── people.txt
├── planets.txt
└── ...
```

Проект состоит из двух основных классов.
- APIRequester
    - Базовый класс для выполнения HTTP-запросов.

- SWRequester
    - Наследует APIRequester и реализует методы работы с API SWAPI.

Также реализована функция save_sw_data(), которая получает список категорий API и сохраняет данные каждой категории в отдельный файл.

**Создать виртуальное окружение**
```bash

```

Windows

